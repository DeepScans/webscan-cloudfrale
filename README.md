# WebScan AI

Умный ИИ-помощник на Cloudflare Workers с поиском и базой знаний.

## Возможности

- 🔍 Поиск через DuckDuckGo Lite
- 📚 Встроенная база знаний (50+ тем)
- 🔐 OAuth вход через Google и Яндекс
- 👤 Гостевой доступ
- 💬 История чатов
- 🎨 Современный тёмный интерфейс
- 📱 Адаптивный дизайн
- ⚡ Edge-вычисления на Cloudflare Workers
